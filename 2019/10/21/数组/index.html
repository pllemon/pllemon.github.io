查找

find() / findIndex()

  • find() 找到第一个返回值为 true 的成员，返回该成员，否则返回 undefined
  • findIndex() 找到第一个返回值为 true 的成员，返回该成员位置，否则返回 -1

填充

fill()

  • fill(填充值*, 起始位置，结束位置)
  • 如填充类型为对象，则是浅拷贝

遍历

entries() / keys() / values()

  • entries() 遍历键值对、keys() 遍历键名、values() 遍历键值
  • 三者皆返回一个遍历器对象，使用 for…of 进行遍历

some()
